# Daily GM on Base ☀️

A simple on-chain GM contract deployed on Base.

## What it does
- Anyone can send **GM**
- One GM per wallet per UTC day
- Tracks total GM count per address
- Fully on-chain and public

## Tech
- Solidity: 0.8.19
- Network: Base Mainnet

## Functions
- `sendGM()` → Send your daily GM
- `gmCount(address)` → Total GM count
- `lastGmDay(address)` → Last GM day

Built for learning and Base Build score.
